🎨 Dynamic Theming

The entire interface adapts based on the --theme variable using CSS if() with style() conditions. Colors, shadows, and effects change automatically.

📏 Responsive Sizing

Font sizes, padding, and border radius scale based on the --size variable. Everything adjusts proportionally using conditional CSS.

🎯 Smart Layout

Layout switches between grid, flex, and block based on the --layout variable. Each layout has optimized spacing and behavior.

✨ Conditional Effects

Animations, glows, and hover effects are conditionally applied based on theme and animation settings using complex if() logic.

background: if( style(--theme: dark): linear-gradient(135deg, #1e1e2e, #2a2a3e); style(--theme: light): linear-gradient(135deg, #f8fafc, #e2e8f0); style(--theme: neon): linear-gradient(135deg, #0f0f23, #1a1a2e); else: #ffffff ); font-size: if( style(--size: small): 2rem; style(--size: medium): 3rem; style(--size: large): 4rem; else: 2.5rem ); animation: if( style(--animation: enabled) and style(--theme: neon): glow 2s ease-in-out infinite alternate; else: none );
