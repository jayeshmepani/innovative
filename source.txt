Transformer.js is not limited to sentiment analysis; it's a versatile library that supports a wide range of machine
learning tasks across natural language processing (NLP), computer vision, audio processing, and multimodal applications.
It leverages pre-trained transformer models from the Hugging Face Hub, running them directly in the browser via
WebAssembly and ONNX Runtime.

### List of All Supported Purposes (Tasks/Pipelines)
Based on the official documentation, here is a complete list of supported tasks/pipelines in transformers.js (as of the
latest updates). I've categorized them for clarity, including brief descriptions of their purposes. Note that some tasks
may require specific model configurations or hardware (e.g., WebGPU for faster inference on larger models), and not all
models work perfectly on all browsers—Chrome with experimental flags enabled is recommended for best performance.
Unsupported tasks like table-question-answering are not included here.

#### Natural Language Processing (NLP)
1. **Fill-Mask**: Predicts missing words in a sentence (e.g., cloze tasks for text completion).
2. **Question Answering**: Extracts answers from a given context based on a question.
3. **Sentence Similarity**: Measures semantic similarity between two or more texts (e.g., for search or clustering).
4. **Summarization**: Generates concise summaries of longer texts.
5. **Text Classification (includes Sentiment Analysis)**: Classifies text into categories, such as positive/negative
sentiment or topics.
6. **Text Generation**: Generates new text based on a prompt (e.g., for chatbots or storytelling).
7. **Text-to-Text Generation**: Transforms input text to output text (e.g., paraphrasing, style transfer).
8. **Token Classification (e.g., NER)**: Labels individual tokens in text, such as named entities (persons, locations).
9. **Translation**: Translates text from one language to another.
10. **Zero-Shot Classification**: Classifies text into user-defined labels without specific training.
11. **Feature Extraction**: Generates embeddings (numerical representations) from text for downstream tasks like
similarity search.

#### Computer Vision
12. **Depth Estimation**: Estimates depth maps from images (e.g., for 3D reconstruction).
13. **Image Classification**: Classifies images into categories (e.g., identifying objects or scenes).
14. **Image Segmentation**: Segments images into regions (e.g., pixel-level classification).
15. **Image-to-Image**: Transforms images (e.g., style transfer or enhancement).
16. **Object Detection**: Detects and localizes objects in images with bounding boxes.
17. **Zero-Shot Image Classification**: Classifies images into arbitrary labels without training.
18. **Zero-Shot Object Detection**: Detects objects based on text descriptions without prior training.

#### Audio Processing
19. **Audio Classification**: Classifies audio clips (e.g., sound events or music genres).
20. **Automatic Speech Recognition (ASR)**: Transcribes spoken audio to text.
21. **Zero-Shot Audio Classification**: Classifies audio into user-defined categories.

#### Multimodal (Combining Text, Image, Audio, etc.)
22. **Document Question Answering**: Answers questions about document images (e.g., scanned forms).
23. **Image-to-Text**: Generates text descriptions from images (e.g., captioning).
24. **Text-to-Image**: Generates images from text prompts (requires diffusion models; may need WebGPU).
25. **Text-to-Speech**: Synthesizes speech from text (audio output).
26. **Visual Question Answering**: Answers questions about images.
27. **Reinforcement Learning**: Supports RL models for decision-making tasks (less common in browser).

#### Tabular Data
28. **Tabular Classification**: Classifies tabular data (e.g., rows in a dataset).
29. **Tabular Regression**: Predicts numerical values from tabular data.

These tasks cover thousands of models from the Hugging Face Hub. For each, you can specify custom models. Limitations:
Some (e.g., text-to-image) may be slow without WebGPU; audio tasks require microphone access; large models can take time
to download (100MB+).

<body>
    <div class="container">
        <h1>Sentence Similarity Demo</h1>
        <p>Enter two sentences to compare similarity.</p>
        <textarea id="sentence1" placeholder="First sentence..."></textarea>
        <textarea id="sentence2" placeholder="Second sentence..."></textarea>
        <button id="analyzeButton">Calculate Similarity</button>
        <div id="output" class="loading">Output will appear here...</div>
    </div>

    <script type="module">
        import { pipeline } from 'https://cdn.jsdelivr.net/npm/@xenova/transformers@2.17.2';
        const outputDiv = document.getElementById('output');
        const analyzeButton = document.getElementById('analyzeButton');
        const sentence1 = document.getElementById('sentence1');
        const sentence2 = document.getElementById('sentence2');
        let taskPipeline = null;
        async function loadModel() {
            outputDiv.textContent = 'Loading model...';
            try {
                taskPipeline = await pipeline('feature-extraction', 'Xenova/all-MiniLM-L6-v2'); // Used for similarity via embeddings
                outputDiv.textContent = 'Model loaded!';
            } catch (error) {
                outputDiv.textContent = `Error: ${error.message}`;
            }
        }
        function cosineSimilarity(vecA, vecB) {
            let dotProduct = 0.0;
            let normA = 0.0;
            let normB = 0.0;
            for (let i = 0; i < vecA.length; i++) {
                dotProduct += vecA[i] * vecB[i];
                normA += vecA[i] ** 2;
                normB += vecB[i] ** 2;
            }
            return dotProduct / (Math.sqrt(normA) * Math.sqrt(normB));
        }
        async function performTask() {
            const s1 = sentence1.value.trim();
            const s2 = sentence2.value.trim();
            if (!s1 || !s2) return outputDiv.textContent = 'Enter both sentences.';
            if (!taskPipeline) return outputDiv.textContent = 'Model loading...';
            outputDiv.textContent = 'Processing...';
            try {
                const emb1 = await taskPipeline(s1, { pooling: 'mean', normalize: true });
                const emb2 = await taskPipeline(s2, { pooling: 'mean', normalize: true });
                const similarity = cosineSimilarity(emb1.data, emb2.data);
                outputDiv.textContent = `Similarity Score: ${similarity.toFixed(4)}`;
            } catch (error) {
                outputDiv.textContent = `Error: ${error.message}`;
            }
        }
        analyzeButton.addEventListener('click', performTask);
        loadModel();
    </script>
</body>

</html>

    </div>

    <script type="module">
        import { pipeline } from 'https://cdn.jsdelivr.net/npm/@xenova/transformers@2.17.2';
        const outputDiv = document.getElementById('output');
        const analyzeButton = document.getElementById('analyzeButton');
        const imageUpload = document.getElementById('imageUpload');
        const canvas = document.getElementById('depthCanvas');
        const ctx = canvas.getContext('2d');
        let taskPipeline = null;
        async function loadModel() {
            outputDiv.textContent = 'Loading model...';
            try {
                taskPipeline = await pipeline('depth-estimation', 'Xenova/depth-anything-small');
                outputDiv.textContent = 'Model loaded!';

            }

        async function performTask() {
            if (!imageUpload.files[0]) return outputDiv.textContent = 'Upload an image.';
            if (!taskPipeline) return outputDiv.textContent = 'Model loading...';
            outputDiv.textContent = 'Processing...';
            try {
                const image = await new Promise((resolve) => {
                    const img = new Image();
                    img.onload = () => resolve(img);
                    img.src = URL.createObjectURL(imageUpload.files[0]);
                });
                const result = await taskPipeline(image);
                // Simple visualization: draw depth map (grayscale)
                const depth = result.predicted_depth; // Assume it's a tensor
                const width = depth.shape[1];
                const height = depth.shape[0];
                canvas.width = width;
                canvas.height = height;
                const imageData = ctx.createImageData(width, height);
                const data = depth.data;
                const max = Math.max(...data);
                const min = Math.min(...data);
                for (let i = 0; i < data.length; i++) {
                    const val = 255 * (data[i] - min) / (max - min);
                    imageData.data[i * 4] = val;
                    imageData.data[i * 4 + 1] = val;
                    imageData.data[i * 4 + 2] = val;
                    imageData.data[i * 4 + 3] = 255;
                }
                ctx.putImageData(imageData, 0, 0);
                outputDiv.textContent = 'Depth map rendered below.';
            } catch (error) {
                outputDiv.textContent = `Error: ${error.message}`;
            }
        }
        analyzeButton.addEventListener('click', performTask);

        async function startRecording() {
            try {
                const stream = await navigator.mediaDevices.getUserMedia({ audio: true });
                recorder = new MediaRecorder(stream);
                recorder.ondataavailable = (e) => audioChunks.push(e.data);
                recorder.onstop = processAudio;
                recorder.start();
                recordButton.disabled = true;
                stopButton.disabled = false;
                outputDiv.textContent = 'Recording...';
            } catch (error) {
                outputDiv.textContent = `Error accessing microphone: ${error.message}`;
            }
        }
        function stopRecording() {
            recorder.stop();
            recordButton.disabled = false;
            stopButton.disabled = true;
        }
        async function processAudio() {
            if (!taskPipeline) return outputDiv.textContent = 'Model loading...';
            outputDiv.textContent = 'Processing audio...';
            try {
                const blob = new Blob(audioChunks, { type: 'audio/wav' });
                audioChunks = [];
                const audio = await blob.arrayBuffer();
                const result = await taskPipeline(audio);
                outputDiv.textContent = result.text;
            } catch (error) {
                outputDiv.textContent = `Error: ${error.message}`;
            }
