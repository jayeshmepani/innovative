🔮 Advanced Gradient Border Techniques

A deep dive into various methods for creating gradient borders, each with its own use case, from simple to complex, animated, and transparent-friendly solutions.

1. `border-image` (No Radius)

Your frontend doesn't have to be a mess.

Get Your Audit 
.media-object {
  border: 4px solid;
  border-image: conic-gradient(...) 1;
}

2. Pseudo-element (With Radius)

Your frontend doesn't have to be a mess.

Get Your Audit 
.media-object {
  position: relative;
  border: 4px solid transparent;
  border-radius: 24px;
}
.media-object::before {
  content: "";
  position: absolute;
  inset: -4px;
  z-index: -1;
  border-radius: inherit;
  background-image: conic-gradient(...);
}

3. Multiple Backgrounds

Your frontend doesn't have to be a mess.

Get Your Audit 
.media-object {
  border: 4px solid transparent;
  border-radius: 24px;
  background-image: 
    linear-gradient(bg-color, bg-color), 
    conic-gradient(...);
  background-clip: padding-box, border-box;
  background-origin: padding-box, border-box;
}

4. CSS Mask (With Transparency)

Your frontend doesn't have to be a mess.

Get Your Audit 
.media-object::before {
  /* ... positioning ... */
  background-image: conic-gradient(...);
  mask: linear-gradient(#fff, #fff),
        linear-gradient(#fff, #fff);
  mask-clip: content-box, border-box;
  mask-composite: exclude;
}

5. Animated Border (Hover Me)

Your frontend doesn't have to be a mess.

Get Your Audit 
@property --angle-spin {
  syntax: "<angle>";
  initial-value: 0turn;
}
.media-object {
  border-image: conic-gradient(
    from var(--angle-spin), ...
  ) 1;
  animation: spin 3s linear infinite paused;
}
.media-object:hover {
  animation-play-state: running;
}
@keyframes spin {
  to { --angle-spin: 1turn; }
}

🚀 Motion Path Animation with `offset-path`

offset-path defines a geometric path for an element to follow. Combined with offset-distance, we can achieve complex, non-linear movement.
