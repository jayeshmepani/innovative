🌑 Shadow DOM Architecture Deep Dive

Understanding the complete structure and syntax

🏗️ 1. Basic Shadow DOM Structure
// ANATOMY OF SHADOW DOM
const host = document.querySelector('#my-element');   // Shadow Host
const shadowRoot = host.attachShadow({mode: 'open'}); // Shadow Root
shadowRoot.innerHTML = '<p>Shadow content</p>';     // Shadow Tree
Key Components:
• Shadow Host: The regular DOM element that contains the shadow DOM
• Shadow Root: The root node of the shadow tree
• Shadow Tree: The DOM tree inside the shadow root
• Shadow Boundary: Where the shadow DOM ends and regular DOM begins

🔐 2. Shadow DOM Modes
// OPEN MODE (accessible from outside)
const openShadow = element.attachShadow({mode: 'open'});
console.log(element.shadowRoot); // Returns the shadow root

// CLOSED MODE (not accessible from outside)
const closedShadow = element.attachShadow({mode: 'closed'});
console.log(element.shadowRoot); // Returns null

🎨 3. CSS Syntax and Encapsulation
// CSS INSIDE SHADOW DOM
shadowRoot.innerHTML = `
  <style>
    /* :host - styles the shadow host itself */
    :host {
      display: block;
      background: lightblue;
    }
    
    /* :host() - conditional host styling */
    :host(.active) {
      border: 2px solid red;
    }
    
    /* :host-context() - styles based on host's ancestors */
    :host-context(.dark-theme) {
      background: black;
      color: white;
    }
    
    /* Regular styles - only affect shadow content */
    p { color: green; }
    
    /* ::slotted() - styles slotted content */
    ::slotted(span) {
      font-weight: bold;
    }
  </style>
  <p>This paragraph will be green, not red!</p>
  <slot></slot>
`;
Special CSS Selectors:
• :host - Selects the shadow host
• :host(selector) - Host with specific selector
• :host-context(selector) - Host within specific context
• ::slotted(selector) - Styles slotted content

🔌 4. Slot Mechanism

⚡ 5. JavaScript API and Methods
// COMPLETE JAVASCRIPT API
class MyComponent extends HTMLElement {
    constructor() {
        super();
        // Create shadow root
        this.shadow = this.attachShadow({
            mode: 'open',
            delegatesFocus: true // Optional: delegates focus to first focusable element
        });
    }

    connectedCallback() {
        // Called when element is added to DOM
        this.render();
        this.setupEventListeners();
    }

    disconnectedCallback() {
        // Called when element is removed from DOM
        this.cleanup();
    }

    static get observedAttributes() {
        // Which attributes to watch for changes
        return ['data-value', 'disabled'];
    }

    attributeChangedCallback(name, oldValue, newValue) {
        // Called when observed attributes change
        if (name === 'data-value') {
            this.updateValue(newValue);
        }
    }
    
    // Custom methods
    render() {
        this.shadow.innerHTML = `
            <style>/* styles here */</style>
            <div>Content here</div>
        `;
    }

    // Event handling
    setupEventListeners() {
        this.shadow.addEventListener('click', this.handleClick.bind(this));
        
        // Custom events
        this.dispatchEvent(new CustomEvent('componentReady', {
            detail: { component: this },
            bubbles: true
        }));
    }
}

🎯 6. Styling Shadow DOM from Outside
// CSS CUSTOM PROPERTIES (CSS Variables)

// Inside Shadow DOM:
`<style>
    :host {
        /* Define default values */
        --primary-color: blue;
        --text-size: 16px;
    }
    .content {
        color: var(--primary-color);
        font-size: var(--text-size);
        /* Fallback if not provided */
        background: var(--bg-color, white);
    }
</style>`

// From outside (regular CSS):
`<style>
    my-component {
        --primary-color: red;
        --text-size: 20px;
        --bg-color: lightgray;
    }
</style>`

🎯 7. Event Handling and Propagation
// EVENT HANDLING IN SHADOW DOM
this.shadow.addEventListener('click', (e) => {
    console.log('Clicked inside shadow DOM');
    
    // Custom events
    this.dispatchEvent(new CustomEvent('myCustomEvent', {
        detail: { data: 'some data' },
        bubbles: true, // Bubbles through shadow boundary
        composed: true // Crosses shadow boundaries
    }));
});

// Event delegation
this.shadow.addEventListener('click', (e) => {
    if (e.target.matches('.button')) {
        this.handleButtonClick(e);
    }
});

// Focus events
this.shadow.addEventListener('focusin', (e) => {
    console.log('Focus entered shadow DOM');
});

🔍 8. Debugging Shadow DOM
// DEBUGGING TECHNIQUES

// 1. Access shadow roots in console
document.querySelector('my-component').shadowRoot;

// 2. Query inside shadow DOM
document.querySelector('my-component').shadowRoot.querySelector('.internal-element');

// 3. Check if element has shadow root
element.shadowRoot !== null;

// 4. Get all slotted elements
const slot = shadowRoot.querySelector('slot');

DevTools Tips:
• Open DevTools → Settings → Preferences → Elements → Show user agent shadow DOM
• Use $0.shadowRoot in console to access selected element's shadow
• Shadow DOM appears as #shadow-root in Elements panel
• Use "Pierce" option in Elements search to search inside shadow DOM
