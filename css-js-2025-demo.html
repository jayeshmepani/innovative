10 Animate to auto Chrome 129+

Intrinsic Height Animations

Animating from height: 0 to height: auto — finally native, no JS measurement required.

What is CSS @function?
@function lets you define typed, reusable functions directly in CSS. Unlike custom properties, functions can take parameters, include conditional logic via if(), and compute a single return value via the result descriptor.
What is the Temporal API?
Temporal is the long-awaited replacement for JavaScript's broken Date object. It provides immutable date/time objects, first-class timezone support, calendar-aware arithmetic, and clear separation of concepts like PlainDate, ZonedDateTime, and Instant.
When is Iterator.prototype available?
Iterator helper methods — .filter(), .map(), .take(), .drop(), .toArray() and more — are part of ECMAScript 2025, officially approved June 25, 2025. They're already shipping in all major engines.

14 Iterator Helpers ES2025

Iterator.prototype Methods

Lazy, chainable transformations on any iterable — no intermediate arrays, works directly on Set, Map, and custom iterators.

// Click Run to execute
const result = ['a', '', 'b', '', 'c', 'd', 'e']
  .values()           // → Iterator
  .filter(x => x)     // skip empty strings
  .drop(1)            // skip first result
  .take(3)            // take only 3
  .map(x => `[${x}]`)  // wrap each
  .toArray();         // → ['[b]','[c]','[d]']

// Also works on Set directly!
const set = new Set([1,2,3,4,5]);
set.values().filter(x => x % 2).toArray();
// → [1, 3, 5]

Safely wraps sync and async code, captures synchronous throws as rejections — bridging mixed-mode code cleanly.

17 RegExp.escape() + Pattern Modifiers ES2025

Safer Regular Expressions

Escape arbitrary strings for use in regex patterns. Plus inline flags via (?i:...) for scoped modifiers.

// Type above to test RegExp.escape()
// Safe dynamic regex
const safe = RegExp.escape(userInput);
const re = new RegExp(safe, 'gi');

// Pattern modifiers (inline flags)
/^prefix(?i:HELLO)suffix$/.test('prefixhelloSuffix')
// true — case-insensitive only within (?i:...)

// Duplicate named groups in alternation
const re2 = /(?<n>\d+)px|(?<n>\d+)em/v;

18 using / await using ES2026 Stage 4

Explicit Resource Management

Automatic cleanup via Symbol.dispose when variables go out of scope — replacing try/finally forever.

// Sync disposal
function withDB() {
  using conn = openConnection();
  // conn[Symbol.dispose]() called automatically
  conn.query('SELECT 1');
} // ← conn closed here, even if error thrown

// Async disposal

  await using file = await openFile(path);
  await file.write(data);
} // ← file.close() awaited automatically
