Pseudo-Elements

New pseudo-elements for styling details content, custom highlights, scroll navigation, backdrops, and text fragments — all native browser features.

::pseudo
::details-content — Animate details expand Chrome 131+ · FF 130+
What is CSS Cascade Layers?
@layer lets you define explicit cascade layers. Lower layers always lose to higher layers, regardless of specificity. Unlayered styles always win.
What is @scope for?
@scope lets you apply styles to a specific DOM subtree (the "scope root") and optionally exclude a sub-subtree (the "scope limit"). Scoped styles have lower specificity by default.
What is anchor positioning?
CSS anchor positioning lets you tether positioned elements to anchor elements. The anchored element reads the anchor's size and position via anchor() and anchor-size() functions, enabling tooltips, dropdowns, and menus without JavaScript.
details::details-content {
  height: 0; overflow: hidden;
  transition: height 0.4s ease,
    content-visibility 0.4s allow-discrete;
}
details[open]::details-content {
  height: auto;
}

line-clamp — Standard property (Chrome 131+)Chrome 131+

Standard line-clamp: 3 (no -webkit- prefix):

The standard line-clamp property finally arrives after years of using the -webkit- prefixed hack. It works with overflow: hidden and clips text at exactly N lines with an ellipsis — no JavaScript needed.

-webkit- compat (still widely needed):

The older -webkit-line-clamp version requires a display:-webkit-box hack. The new standard property is cleaner and more predictable across browsers.

/* Standard — Chrome 131+ */
line-clamp: 3; overflow: hidden;
/* Compat */
display: -webkit-box; -webkit-line-clamp: 3;
-webkit-box-orient: vertical; overflow: hidden;
